# VS Code Theme Generator

Una extensión para Visual Studio Code que permite generar, personalizar y exportar temas personalizados de forma interactiva.

## Características

- Generador de paletas de colores personalizadas
- Editor visual con barra lateral integrada
- Exportación de temas en formato JSON
- Interfaz intuitiva y fácil de usar
- Soporte para temas claro y oscuro

## Requisitos

- Visual Studio Code 1.50.0 o superior
- Node.js 14.0.0 o superior

## Instalación

1. Clone el repositorio
2. Ejecute `npm install` para instalar las dependencias
3. Ejecute `npm run build` o presione `Ctrl+Shift+B` para compilar
4. Presione `F5` para ejecutar la extensión en modo debug

## Uso

1. Abre la paleta de comandos (`Ctrl+Shift+P`)
2. Busca "Theme Generator" para acceder a las funciones principales
3. Personaliza los colores usando la barra lateral
4. Exporta tu tema personalizado

## Configuración

La extensión se configura a través de los ajustes estándar de VS Code. Busca "Theme Generator" en los ajustes para ver las opciones disponibles.
